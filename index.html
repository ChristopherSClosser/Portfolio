My Software Dev Life


  I have always been obsessed with the inner workings of everything. From opening my "etch a sketch" (I shouldn't have to say what a mess that made) at age 8, to now being able to fix nearly anything, always striving to make it better than before.

  It is important for every task to have the proper and/or best tools. Within a few key strokes almost every tool you would possibly need is available. The knowledge base behind this is astronomical and world wide. This means that there is quite a bit of out of date or wrong information and even disinformation.

  In the past year going through the intensive Code Fellows Seattle coding bootcamp, I have learned how to sort through and find the best tools needed, make it work, then make it better.
